CXCountDownLabel
================

Counting down label

##How to use
---

```Objective-C

- (id)initWithStartNumber:(NSInteger)startNumber endNumber:(NSInteger)endNumber countDownHandeler:(CXCountDownHandler)countDownHandeler;

- (void)start;
- (void)pause;
- (void)resume;

```
